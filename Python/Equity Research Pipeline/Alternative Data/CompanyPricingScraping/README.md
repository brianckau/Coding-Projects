CompanyPricingScraping is a tool that scrapes target webpage using beautifulsoup and playwright to return key data, namely pricing of products of companies.

It can be deployed in equity research as a primary research & alternative data analysis.

Key functions include:
1) Price fetching from target website
2) Descriptive & predictive analysis of company pricing

It is an optimal analysis pipeline to address the unit economics buildup and alpha extraction from companies that have a major online sales pipeline.
