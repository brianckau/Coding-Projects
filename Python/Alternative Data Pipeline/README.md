Alternative Data Pipeline stores the following projects:

1) CompanyPricingScraping: Scrape company pricing of products to extract pricing trend horizontally (across competitors) or across a time series for unit economics buildup & alpha extraction.
2) TimeSeriesNewsSentiment: Extract google news sentiment for the top 100 news each day in a given period for a given query; allows linkage and correlation analysis with asset returns.

These tools facilitate primary research for equity pitching processes with alternative data analysis.
